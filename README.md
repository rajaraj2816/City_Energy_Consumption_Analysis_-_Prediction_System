# City_Energy_Consumption_Analysis_-_Prediction_System

A data analytics and machine learning project that analyzes and predicts city-wide electricity consumption using weather conditions, special events, and zone-wise energy data. The project includes synthetic dataset generation, data preprocessing, exploratory data analysis (EDA), interactive visualizations, and predictive modeling using Linear Regression and Random Forest algorithms. The analysis provides actionable insights into energy demand patterns and supports data-driven decision-making for smart energy management.


Key Features
  Synthetic energy consumption dataset generation
  Data cleaning and preprocessing
  Exploratory Data Analysis (EDA)
  Interactive visualizations using Plotly
  Energy consumption forecasting
  Linear Regression & Random Forest models
  Business insights for smart energy planning
